# Transport Route Management System

A web-based application designed to streamline, schedule, and  transport routes. This system provides an intuitive interface for managing transit lines, schedules, and vehicle allocations efficiently.

## 🚀 Tech Stack

- **Backend:** Java
- **Frontend:** HTML, JSP (JavaServer Pages)
- **Database:** Oracle Database
- **Architecture:** DAO (Data Access Object) Pattern & Java Bean Classes

## 🏗️ Architecture & Design

The project follows a clean, decoupled architecture to ensure scalability and maintainability:
- **Bean Classes:** Represent data models (e.g., Route, Vehicle, Schedule) with standard encapsulation (getters/setters).
- **DAO Pattern:** Separates the core business logic from the database persistent layer, handling all Oracle SQL operations cleanly.
- **JSP & HTML:** Provides a dynamic user interface to interact with the backend services seamlessly.

## ⚙️ Prerequisites

Before running the project, ensure you have the following installed:
- JDK 21 or higher
- Oracle Database (Express Edition or Enterprise)
- An Enterprise IDE (e.g., Eclipse IDE for Enterprise Java developers or IntelliJ IDEA)
- Apache Tomcat Server (or any compatible servlet container)
